
# Tic-Tac-Toe Game

A simple and interactive Tic-Tac-Toe game built using **HTML, CSS, and JavaScript**. The game supports two players, detects winners automatically, and handles draw situations.

## Features

* Two-player gameplay (X and O)
* Automatic winner detection
* Draw/Tie detection
* Reset Game functionality
* New Game functionality
* Responsive and modern UI
* Disabled cells after selection to prevent overwriting moves

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How to Run

1. Clone the repository:

```bash
git clone https://github.com//tic-tac-toe.git
```

2. Navigate to the project directory:

```bash
cd tic-tac-toe
```

3. Open `index.html` in any web browser.

No additional dependencies or installations are required.

## Game Rules

1. The game is played on a 3×3 grid.
2. Player O always starts first.
3. Players take turns placing their marks (X or O).
4. The first player to align three marks horizontally, vertically, or diagonally wins.
5. If all cells are filled and no player wins, the game ends in a draw.

## Winning Combinations

```
[0,1,2]
[0,3,6]
[0,4,8]
[1,4,7]
[2,5,8]
[2,4,6]
[3,4,5]
[6,7,8]
```
